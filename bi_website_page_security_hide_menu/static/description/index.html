Website Menu,Sub Menu and Page Security Odoo App

Website page and menu security odoo app helps user to hide any menu, sub-menu, pages, from internal, portal or public user. You can also hide menu, pages, submenu for specific user, specific groups. Using this app user can also apply website page login security for users.

Features

Hide Any Menu, Sub-Menu, Page.

User can hide any menu, sub-menu, page of website.

Hide from Internal, Public and Portal Users.

User can hide menu for internal, public and portal user types.

Hide Menu from Specific Users.

User can hide menu, sub-menu, pages from specific users.

Hide Menu from Specific Groups

User can hide menu, sub-menu, pages from specific groups.

Hide Menu Option Available on Users, Groups, Pages.

User have option to hide from generic menu, user form view, groups from view and pages.

Website Login Pages Security.

User can apply login access security on website.
Demo Request Video Tutorial

Website Page Security

User can restrict page for public user, User just have to enable "User Login Required ?" on website page.

User can see that without login (Public User) webpage can not accessible.

Hide Website Any Menu, Sub-Menu, Page

User can hide website any menu, sub-menu or page.

Hide Menu, Sub-Menu, Page Configuration

Under Website > user can see "Generic Hide Menu Config" menu here user can hide menu, sub-menu or page.

User can hide menu, sub-menu or page for every Internal, Public or Portal user.

Hide Menu, Sub-Menu or Page for Internal Users

User can see visible menus on website page.

User can also see visible page of website.

User can see added menu and webpage to hide from internal user.

User can see selected menus are hidden for internal users.

User can also see that selected webpages also hidden for internal user.

Hide Menu, Sub-Menu or Page for Portal Users

User can see visible website menus.

User can also see visible webpage for portal users.

User can see added menu and webpage to hide from internal user.

User can see selected menus are hidden from portal users.

User can also see that selected webpages also hidden from portal users.

Hide Menu, Sub-Menu or Page for Public Users

Use can see visible menus on webpage for public users.

User can see selected menus to hide for public users.

User can see selected menus are hidden from website for public users.

Hide Menu, Sub-Menu, Page from User Setting

User can configure to hide menu also from user settings under Settings > User & Companies > Users on "Hide Menus" tab.

User can also hide page also from user settings under "Hide Pages" tab.

Hide Menu, Sub-Menu, Page from Groups

User have options to hide menu, sub-menu, page from particular groups, On group form view user can see "Hide Menu" tab, from here user can hide menu.

User can also hide page from particular group under "Hide Pages" tab.

Hide from Page Form View

User can also hide from page from view, user can see tab "Hide From Users" and "Hide From Groups" tab.
